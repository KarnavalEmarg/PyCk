# PyCk

For complete details please take a look at the [wiki](https://github.com/Naategh/PyCk/wiki).

## Installation

- git clone https://github.com/Naategh/PyCk.git
- cd PyCk
- pip3 install -r requirements.txt

## Install via docker
`git clone https://github.com/Naategh/PyCK.git`

`docker build .`

## Requirements
- Python 3.5
